# My base camp project for [qwasar.io](https://upskill.qwasar.io)
## Expressjs & sqlite3 project


### install this project
```
  npm install && npm i nodemon -g
```

### Running server for development
```
  npm start dev:start
```



### Run server for review
```
  npm start
```
